A computer memory is basically divided in two: Primary (RAM) and Secondary (Flash drives, DvD, CD-ROMs).

The main memory is called RAM (Random Access Memory). It consists of collection of addresses and contents. An address refers a byte of memory which can be of any data type like integers, string, etc (1 byte = 8 bits(1's and 0's)).

Addresses are usually represented using hexadecimal base-16 counting system. And they are prefixed with 0x to indicate they are of a hexadecimal type
