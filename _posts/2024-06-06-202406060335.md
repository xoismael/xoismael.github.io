In c we can evalute booleans just by only using numbers. The number 0 is false. And any other number is automatically true.

Example:

```c
int num = 1;
if (num) {
    
}
```
The above code runs whatever is in the conditional block since it evalutes to true. (It is true because any number other than 0 is considered as true.)
